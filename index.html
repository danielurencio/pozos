Práctica: Análisis de datos para la exploración de pozos¶

Resumen¶

El objetivo de este ejercicio es determinar, a travs de un anlisis de datos, en qu pozos debe de enfocarse la atencin para la extraccin de petrleo. La metodologa adoptada en este documento consiste en analizar la serie de tiempo para cada uno de los pozos con datos para 14 meses. Es decir, slo se consideran aquellos pozos que cuenten con informacin para el periodo comprendido entre enero de 2016 y febrero de 2017; dicho rango es el mximo posible de acuerdo con la base de datos proporcionada.

Para cada uno de los pozos que cumpli con el criterio descrito anteriormente se ajust una regresin lineal cuya variable dependiente $y$ es la produccin de petrleo, y la variable independiente $x$ es el ndice del periodo observado. Debido a que se cuenta con 14 meses, el ndice de cada periodo es un nmero entero entre 0 y 13 ( $x \in [0,13]$ ). Cada una de estas regresiones se ajust para fines meramente descriptivos pues su propsito es obtener una lnea de tendencia durante los 14 periodos analizados. Para conocer si la tendencia es positiva o negativa se observ el valor de la pendiente resultante del cmputo de una funcin lineal a travs del mtodo de mnimos cuadrados ordinarios; y para conocer si dicha tendencia, o comportamiento, ha sido estable a lo largo del periodo analizado se observ el coeficiente de determinacin $R^2$.

Los pozos seleccionados fueron aquellos cuya lnea de tendencia fue positiva y cuya bondad de ajuste es mayor o igual a 0.6; 298 pozos cumplen con esta condicin.

Esta prctica se desarroll con el lenguaje de programacin Python y este documento se hizo con una herramienta llamada Jupyter. Este trabajo est dividido en tres secciones: procesamiento de datos, metodologa, y consideraciones finales.

2.2 Parámetros de una regresión lineal simple como medida de tendencia¶

De manera descriptiva, pueden utilizarse algunos parmetros obtenidos a travs del mtodo de mnimos cuadrados ordinarios para establecer medidas de tendencia para una serie de tiempo. Convencionalmente, una regresin lineal, en su forma ms simple, est dada por:

$$\hat{y} = \theta_0 + \theta_1 \cdot x$$

En donde $\hat{y}$ es el valor estimado, o variable dependiente, $x$ es la variable independiente, y los parmetros $\theta_0$ y $\theta_1$ representan el intercepto y la pendiente de la funcin ajustada respectivamente.

Asimismo, el coeficiente de determinacin, o $R^2$, es una medida que ofrece informacin sobre la proporcin de la varianza de la variable dependiente que es explicada por la variable independiente; en otras palabras, este coeficiente es una medida para evaluar la bondad del ajuste de una regresin lineal. Este coeficiente, para una ecuacin con una sola regresora, puede obtenerse al elevar al cuadrado el coeficiente de correlacin de Pearson.

$$ \rho^2 = \left( \frac{cov(x,y)}{\sigma_x \cdot \sigma_y} \right) ^2\ = R^2 \in [0,1] $$

Para obtener una medida descriptiva de tendencia para cada una de las series de tiempo de los pozos seleccionados, se estimarn las pendientes y los coeficientes de determinacin, $\theta_1$ y $R^2$.

El signo del valor de la pendiente $\theta_1$ ser interpretado como un indicador para determinar si la produccin de un pozo se ha mantenido constante, si ha crecido, o si ha decrecido. Asimismo, el coeficiente de determinacin $R^2$ se utilizar como una medida para conocer si el comportamiento de la tendencia es estable; mientras ms cercano a $0$ sea el valor del coeficiente ms inestable es la tendencia, y mientras ms cercano a $1$ ms estable es sta.

$$ \theta1 < 0 \rightarrow tendencia\ a\ la\ baja $$$$ \theta1 > 0 \rightarrow tendencia\ a\ la\ alza $$$$ R^2 \approx 0 \rightarrow tendencia\ inestable $$$$ R^2 \approx 1 \rightarrow tendencia\ estable $$

En otras palabras, para cada pozo se crear una instancia del objeto, de nombre Reg (por regresin lineal), y cada instancia podr ejecutar los mtodos que estimarn $\theta_1$ y $R^2$ de la serie de tiempo que corresponden al pozo asignado al objeto. class Reg(object):
    def __init__(self,pozo,variable):
        self.pozo = pozo
        self.variable = variable
        self.x = 0
        self.y = 0
        self.slope = 0
        self.intercept = 0
        self.R_2 = 0
        self.y_yat = 0

    def Parametros(self):
        self.y = data[data['pozo'] == self.pozo][[self.variable]].values;
        self.y = self.y.reshape(1,self.y.shape[0])[0].tolist();
        self.x = [];
        for i in range(len(self.y)):
            self.x.append(i);


        self.x = np.array(self.x);
        self.y = np.array(self.y);
        self.slope,self.intercept,self.R_2,p_value,std_err = stats.linregress(self.x,self.y);
        self.R_2 = self.R_2**2

    def Graficar(self):
        self.y_hat = self.intercept + self.slope * self.x
        plt.plot(self.x,self.y)
        plt.plot(self.x,self.y_hat)
        plt.show()

    def Parametros(self):
        # Se almacenan los datos de la variable seleccionada, para el pozo analizado,
        # y se utilizan como la "y", o variable dependiente, de la regresin.
        self.y = data[data['pozo'] == self.pozo][[self.variable]].values;
        self.y = self.y.reshape(1,self.y.shape[0])[0].tolist();
        # Como variable independiente se utilizan los ndices de los periodos existentes.
        # Dado que se toman 14 periodos, el ndice para el primer mes sera el 0, para el
        # segundo mes sera el 1, para el tercero el 2, y as sucesivamente hasta llegar al 13.
        self.x = [];
        for i in range(len(self.y)):
            self.x.append(i);


        self.x = np.array(self.x);
        self.y = np.array(self.y);
        # A travs de la funcin "stats.linregress" de la librera 'scipy' pueden obtenerse
        # cinco parmetros de una regresin lineal simple: la pendiente, el intercepto, el
        # coeficiente de Pearson, el 'p value' y el error estndar.
        self.slope,self.intercept,self.R_2,p_value,std_err = stats.linregress(self.x,self.y);
        # Para obtener el coeficiente de determinacin, R^2, simplemente se eleva al cuadrado
        # al coeficiente de Person obtenido en la funcin anterior.
        self.R_2 = self.R_2**2
